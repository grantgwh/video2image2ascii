动画视频转换为字符动画（opencv重写版）
========
使用的相关软件：
python3
opencv2.4
numpy
--------
代码原理：
:按帧读取视频文件
:缩小帧的像素
:将每一帧转换为灰度图片
:设置一个黑白分割阀值（我这里使用127作为阀值）
:小于阀值的用‘＃’代表黑色，大于阀值的用‘ ’代表白色
:将替换后的字符写入文本并且排序
:搭配使用'cat','clear'(linux)--'cls'(windows),'time.sleep'实现文本在终端滚动播放


PS:第一次运行需要在项目根目录手动建立‘text’文件夹，先运行‘Video_Deal.py’，再运行’play.py‘
